# Digital Clock

This is a simple digital clock application built using Python and Tkinter. It provides a user-friendly graphical interface to display the current time digitally.

## Features

- Display the current time in digital format.
- Simple and intuitive user interface.

## Requirements

- Python 3.x
- Tkinter

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/jmugambwa1/Digital-clock.git
    ```

2. Navigate to the project directory:

    ```bash
    cd Digital-clock
    ```

3. No additional dependencies need to be installed as Tkinter comes pre-installed with Python.

## Usage

1. Run the following command:

    ```bash
    python main.py
    ```

2. The digital clock GUI will open.
3. The current time will be displayed in the format HH:MM:SS.
4. The clock updates in real-time to reflect the current time.
